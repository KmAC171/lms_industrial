# 🚀 Advanced Learning Management System (LMS)

A modern, scalable, AI-powered Learning Management System (LMS) built for schools, universities, corporate training, and online education platforms.

# ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/yourusername/lms-project.git

# Navigate into project
cd lms-project

# Install dependencies
npm install

# Run development server
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file:

```env
DATABASE_URL=
JWT_SECRET=
NEXTAUTH_SECRET=
NEXT_PUBLIC_API_URL=
AWS_ACCESS_KEY=
AWS_SECRET_KEY=
```
